# crystal-random

This library contains utilities to generate random numbers.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  crystal-random:
    github: crystal-lang/crystal-random
```

## Usage

```crystal
require "crystal-random"
```
